IP-XACT import / export package

## Use

### DUH -> SPIRIT

```
npx duh-ipxact duh2spirit myDuh.json5 mySpirit.xml

```

### IPXACT -> DUH

```
npx duh-ipxact ipxact2duh myIpxact.xml myDuh.json5

```
